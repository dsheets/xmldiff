default_costs [Xmldiff]
Default cost functions, defined as: cost_insert size _ = size, cost_delete size _ = max (size / 4) 1, cost_edit (label1, _) (label2,_) = if label1 = label2 then 0 else 1 By defining cost functions, you can tune the precision of operations.
